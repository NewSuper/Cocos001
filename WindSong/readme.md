## 游玩指引

- 键盘按键
  
  `Q` `W` `E` `R` `T` `Y` `U`
  
  `A` `S` `D` `F` `G` `H` `J`

  `Z` `X` `C` `V` `B` `N` `M`

  对应每一行琴键  _（或者直接点击琴键）_
  
- 键盘按键 `O` 开始/停止自动弹奏 _（或者点击左下角按钮）_

- 键盘按键 `P` 设置自动弹奏延时  _（或者点击左下角按钮）_
